# 实验报告：Log4j2 44228漏洞环境搭建、漏洞验证与漏洞利用复现
## 一、实验目的
    搭建Log4j2 44228漏洞环境。
    验证漏洞的存在。
    利用漏洞获得flag

## 二、实验环境

### 操作系统：
1. Kali Linux
2. Java版本：JDK 1.8.0 202
3. Log4j2:
4. 测试工具：JNDI injector
